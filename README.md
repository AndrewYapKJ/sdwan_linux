# SD-WAN Linux Project

This project provides a Linux-based SD-WAN implementation using IPsec tunnels.

## Project Structure

- `basic_setup/`: Basic SD-WAN setup with IPsec, bandwidth control, and routing. Suitable for simple deployments.
  - Includes aggregator and CPE configurations, setup scripts, and beginner-friendly docs.
- `advanced/`: Advanced features for production use, such as high availability, monitoring, security enhancements, etc.

## Getting Started

Start with `basic_setup/README_Beginner.md` if you're new to this.

For testing with Parallels and cloud: See `basic_setup/TESTING_GUIDE.md`.

For production, check `advanced/` for enhancements.
